## 当前定位

Hermes-Yachiyo 是桌面优先的本地个人 Agent 应用，不是普通聊天网页。它把 Hermes Agent 包装成本机常驻助手，并提供多个共享同一会话状态的入口：

- 主控台：配置、状态、工具、维护入口。
- Chat Window：完整对话窗口。
- Bubble：轻量悬浮入口。
- Live2D：角色桌面入口。
